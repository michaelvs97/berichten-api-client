# Org.OpenAPITools.Model.BerichtKenmerken
De BerichtKenmerken lijkt op de MessageHeading zoals dat gebruikt werd bij de mailboxserver. Ten opzichte van de mailboxserver zijn de onderstaande velden niet langer aan de orde en sommige hebben een andere naam gekregen.   * CrossReference - -> verwijzingBerichtId   * MessageId - -> berichtId   * NotificationRequest: Wordt niet door deze API ondersteund.   * NumberOfRecipients: Wordt niet door deze API ondersteund.   * RecipientORName - -> ontvanger
